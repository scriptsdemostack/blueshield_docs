App settings

Blueshield provides simple yet powerful interface for manage global Blueshield app settings. it allows administrators to easily enable/disable or change Blueshield global settings.

General app settings

Blueshield's general app settings page can be used to update Blueshield application name and email sending email address.

User app settings

Blueshield's user app settings page can be used to enable/disable system features or change system variables on Blueshield application.
