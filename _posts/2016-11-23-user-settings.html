User settings

Blueshield provides simple yet powerful interface for users to manage their user settings. it allows users to change their profile info and change user password.

User profile settings

Blueshield's user profile settings page can be used to update users profile page info.

User account settings

Blueshield's user account settings page can be used to change users user password.

User localization settings

Blueshield's user localization settings page can be used to change users language.
